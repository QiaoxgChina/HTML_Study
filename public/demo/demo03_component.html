创建组件

第一种方式：继承 React.Component

    
        class HelloMessage extends React.Component {
            render() {
                return Hello {this.props.name};
            }
        }

第二种方式：函数方式

        
            function HelloMessage(props) {
                return World 111111111111, {params.name};
            }
